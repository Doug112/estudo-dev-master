Elementos genéricos não semânticos

Escrever HTML semântico é algo que leva tempo, e precisa de muita prática para fazer com que o HTML tenha maior significado. Temos dois elementos não semânticos para uso genérico, um para bloco e outro para texto, sendo esses o div e o span. Em ambas as situações nós usaremos atributos globais como id e class para entregar um maior significado. O span é usado quando não se acha um elemento de texto semântico, já o div, quando não se acha um elemento de bloco semântico.
